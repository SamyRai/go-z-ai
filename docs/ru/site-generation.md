# Генерация сайта

Проект поставляет автоматически сгенерированный статический HTML-сайт в `site/`,
созданный из markdown-документации и живых данных GitHub небольшим
Go-генератором.

Полное описание архитектуры и инструкция по добавлению новых локалей — в
[английской версии](../en/site-generation.md).

## Запуск

```bash
make site          # сгенерировать в ./site
make site-offline  # то же, но без обращений к GitHub API
make site-serve    # сгенерировать + отдавать на http://localhost:8000
```

## Что используется

- **goldmark** для рендеринга markdown в HTML (уже в `go.sum` через glamour).
- **Только стандартная библиотека Go** в остальном.
- **Тема Catppuccin Mocha/Latte** (MIT), уже в `go.sum` через `catppuccin/go`.
- Никакого JS-фреймворка, веб-шрифтов или внешних CSS-зависимостей.
